# Documentation Generator Expert

This skill specializes in transforming raw code logic into human-readable, professional technical documentation.

## Capabilities

- **Architecture Mapping**: Translates imports and exports into Mermaid dependency diagrams.
- **Logic Distillation**: Explains complex `useEffect` or state synchronization logic in simple terms.
- **README Generation**: Crafts high-impact project or component READMEs with a focus on "Premium" aesthetics.

## Rules

1. **No Jargon**: Avoid over-complicating explanations unless specifically asked for deep technical detail.
2. **Visuals First**: Use Mermaid for every complex workflow.
3. **Sentences Over Fragments**: Use full, professional sentence case for all headings and descriptions.
4. **Integration**: Always identify how the component fits into the broader Lemon ecosystem (e.g., Auth, UI Context).
