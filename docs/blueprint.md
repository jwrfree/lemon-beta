# **App Name**: Lemon

## Core Features:

- Transaction Logging: Log income and expense transactions, including amounts, categories, and descriptions.
- Wallet Management: Allow users to manage multiple wallets (e.g., bank accounts, cash, credit cards).
- Categorization of transactions: Offer the user the opportunity to categorize a transaction during transaction entry
- Transaction Summaries: Use an LLM tool to generate a brief text summary of spending and income transactions in a selected category, suitable for journaling or reflection.
- Budget Setting: Allow users to set budgets for different spending categories.
- Spending/Income Visualization: Present the spending/income data with charts, with breakdowns by wallet or category
- Data Import/Export: Allow users to import and export transaction data in CSV format.

## Style Guidelines:

- Primary color: White (#FFFFFF) for a clean and modern look.
- Secondary color: Light gray (#F0F0F0) for backgrounds and subtle UI elements.
- Accent color: Teal (#008080) to highlight important actions and information.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern appearance.
- Use minimalist line icons from a set like 'Feather' for a consistent and clean aesthetic.
- Employ a card-based layout to organize information clearly, particularly on dashboard and summary screens.
- Incorporate subtle fade and slide transitions when navigating between pages or updating data.
